Como funciona?
1. Computador faz uma requisição
2. Servidor processa
3. Servidor devolve o HTML
4. Navegador apresenta o HTML

Como de programa para internet?

Através de:

* Linguagem do cliente:
- HTML: linguagem de marcação de dados
- CSS: folha de estilos
- JS: script no cliente

* Linguagem de servidor:
- PHP
- ASP
- JSP
- COULDFUSION
- PYTHON
- PEARL

Para que serve o HTML?
Ele que possibilita que a informação da página seja apresentada pelo navegador.

Por onde começar?
1. Ter um navegador
2. Um bloco de notas
